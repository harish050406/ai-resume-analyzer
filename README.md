# AI Resume Analyzer

AI-powered web app to match resumes with job descriptions using NLP and Machine Learning.

## How It Works

![Workflow](workflow.png)

##  Features
- Resume parsing (PDF)
- Skill extraction
- Match score using TF-IDF
- Skill gap analysis
- Resume feedback

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- PyPDF2

## Run Locally
```bash
streamlit run app.py
